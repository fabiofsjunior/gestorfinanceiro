Introdução

A linguagem HTML5 basicamente funciona com um editor de textos, baseados em TAGS, as tags são utilizadas para fazer marcações no texto, dando assim sentido ao conteúdo, ela não é conhecida com uma linguagem de programação. pois a mesma não realiza nenhum tarefa computacional, fazendo jus ao seu nome Hypertext Markup Language. Com ela podemos fazer textos simples, listas, tabelas, campo de inserção de dados, ancoragem de links, adicionar arquivos locais e externos (url, imagens, videos e audios) etc. Contudo apenas texto estático, e com o apoio do CSS3 e a linguagem JavaScript podemos desenvolver sites simples e rebuscados explorando seus recursos, este simples site tem como objetivo trazer um breve resumo do que a linguagem é capaz de um nível mais básico para avançar e se tornar mais profissional procure outras fontes mais especifícas.

Comandos Básicos para textos

O texto mais estruturado é composto por títulos e parágrafos,esteja você lendo uma história, um jornal, um livro da faculdade, uma revista, etc. O conteúdo estruturado torna a experiência de leitura mais fácil e mais agradável. Em HTML, cada parágrafo deve ser envolvido em um elemento <p>, assim:

<p>Eu sou um parágrafo, oh sim, eu sou.<p>
